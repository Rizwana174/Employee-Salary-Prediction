# Employee Salary Prediction

A Machine Learning web application that predicts whether an employee's salary is likely to be greater than 50K or less than/equal to 50K based on employee details.

## Technologies Used
- Python
- Flask
- Scikit-Learn
- HTML
- CSS
- GitHub

## Features
- User-friendly web interface
- Salary prediction using KNN Machine Learning model
- Real-time prediction

## Project Structure
- app.py
- templates/index.html
- static/style.css
- model/salary_model.pkl
